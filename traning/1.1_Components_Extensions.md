# Quarkus Components (Extensions) — Use Cases & Examples

## 1. RESTEasy Reactive (`quarkus-resteasy-reactive`)
**Use case:** Build REST APIs (JAX-RS based).

```java
@Path("/users")
public class UserResource {
    @GET
    @Produces(MediaType.APPLICATION_JSON)
    public List<User> getUsers() {
        return User.listAll();
    }
}
```

---

## 2. Hibernate ORM with Panache (`quarkus-hibernate-orm-panache`)
**Use case:** Simplified JPA/ORM data access (active record pattern).

```java
@Entity
public class User extends PanacheEntity {
    public String name;
    public String email;
}

// Usage
User.find("name", "John").firstResult();
User.listAll();
```

---

## 3. Hibernate Reactive with Panache (`quarkus-hibernate-reactive-panache`)
**Use case:** Non-blocking, reactive database access for high-throughput services.

```java
@Entity
public class User extends PanacheEntity {
    public String name;
}

Uni<List<User>> users = User.listAll();
```

---

## 4. Quarkus REST Client (`quarkus-rest-client-reactive`)
**Use case:** Call external REST APIs declaratively.

```java
@RegisterRestClient(baseUri = "https://api.example.com")
public interface ExternalApi {
    @GET
    @Path("/data")
    Response getData();
}
```

---

## 5. SmallRye Kafka (`quarkus-smallrye-reactive-messaging-kafka`)
**Use case:** Event-driven microservices, produce/consume Kafka messages.

```java
@Incoming("orders-in")
public void process(Order order) {
    // handle event
}

@Outgoing("orders-out")
public Multi<Order> produce() {
    return orderStream;
}
```

---

## 6. SmallRye Health (`quarkus-smallrye-health`)
**Use case:** Kubernetes liveness/readiness probes.

```java
@Liveness
public class PingCheck implements HealthCheck {
    public HealthCheckResponse call() {
        return HealthCheckResponse.up("ping");
    }
}
// Exposed at /q/health/live and /q/health/ready
```

---

## 7. SmallRye OpenAPI / Swagger UI (`quarkus-smallrye-openapi`)
**Use case:** Auto-generate API documentation.

```java
@Tag(name = "Users")
@Path("/users")
public class UserResource { ... }
// UI available at /q/swagger-ui
```

---

## 8. SmallRye Fault Tolerance (`quarkus-smallrye-fault-tolerance`)
**Use case:** Resilience patterns — retries, circuit breakers, timeouts, bulkheads.

```java
@Retry(maxRetries = 3)
@CircuitBreaker(requestVolumeThreshold = 4, failureRatio = 0.5)
public String callDownstream() {
    return externalApi.getData();
}
```

---

## 9. SmallRye JWT (`quarkus-smallrye-jwt`)
**Use case:** Secure REST endpoints with JWT-based authentication/authorization.

```java
@RolesAllowed("admin")
@GET
@Path("/secure")
public String secureEndpoint() {
    return "Authorized!";
}
```

## 10. OIDC (`quarkus-oidc`)
**Use case:** Integrate with Keycloak / Okta / Azure AD for OAuth2/OpenID Connect login.

```properties
quarkus.oidc.auth-server-url=https://keycloak.example.com/realms/myrealm
quarkus.oidc.client-id=my-app
quarkus.oidc.credentials.secret=my-secret
```

## 11. Quarkus Cache (`quarkus-cache`)
**Use case:** Cache expensive method results in memory (Caffeine).

```java
@CacheResult(cacheName = "user-cache")
public User getUser(Long id) {
    return userRepository.findById(id);
}
```

## 17. Quarkus Vertx / Reactive Streams (`quarkus-vertx`)
**Use case:** Low-level reactive event bus communication between components.

```java
@Inject EventBus bus;

bus.send("order-events", orderPayload);

@ConsumeEvent("order-events")
void onOrderEvent(Order order) { ... }
```

---

## 18. Quarkus gRPC (`quarkus-grpc`)
**Use case:** High-performance service-to-service RPC communication.

```proto
service Greeter {
  rpc SayHello (HelloRequest) returns (HelloReply);
}
```
```java
@GrpcService
public class GreeterService implements Greeter {
    public Uni<HelloReply> sayHello(HelloRequest request) {
        return Uni.createFrom().item(HelloReply.newBuilder()
            .setMessage("Hello " + request.getName()).build());
    }
}
```

---

## 23. Quarkus Redis Client (`quarkus-redis-client`)

```java

```

## 24. Quarkus Native / GraalVM Support (`quarkus-native`)
**Use case:** Compile app to a native binary for fast startup, low memory footprint (ideal for serverless/containers).

```bash
./mvnw clean package -Pnative -Dquarkus.native.container-build=true
```
